# 環境設定

今回必要になるのは、エディタです。

## エディタとは？

コードは【メモ帳】でも書き換えることが可能ですが、エディタを使用することによって、ハイライト（色分け）、エラーチェック、自動生成などを利用して、コーディングを効率的に進めることができます。

様々な種類がありますが、ここでは【無料】で拡張機能も充実しているMicrosoft社のVisual Studio Codeを使います。(通称VSCode"ブイエスコード")

## VSCode
