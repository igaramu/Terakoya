# HTMLに触れてみる

HTMLとは【マークアップ言語】と呼ばれます。

## マークアップ言語とは？

文字列に意味のあるマークを付け、PCに認識してもらうことです。

例えば…
```html
私はアライグマのアラシです。
<br>
年齢は30歳です。
```
<div class="result">
■出力結果
```
私はアライグマのアラシです。
年齢は30歳です。
```
</div>

`<br>`がタグになります。
`<br>`は【改行】を意味します。

また、`<br>`のように単体で使うものや、`<p> ** </p>`のように`/`で閉じるものがありあす。

以下は特によく使います。

- \<h1> ~~ \</h1> = 見出し１
- \<p> ~~ \</p> = 段落
- \<ul> ~~ \</ul> = 箇条書き
